# OmniFile

Hello this is my python project called OmniFile. Omni meaning "all" so basically it converts between the major file types like images, audio, csv, etc. 

For this program all you need to do is download the .exe from the github release page right in this repo and you can attach a file and convert it to another file type. 
I made this because converting files online is soooo annoying and websites online usually make you watch ads or are filled with ads and are very slow and have a limit on file size and a hard cap of how many a day you can do.  

So therefore I made my own software .exe that can do all of this locally on your computer, so it's 100% safe, offline, open source, and also unlimited and istaneous with no ads. 

Here are supported file conversions:

  Images: JPG, PNG, WEBP
  
  Audio: MP3 <-> WAV
  
  Video: MP4 -> MP4, MP3, WAV
  
  CSV <-> XLSX
  
  CSV <-> JSON
  
  XLSX <-> CSV
  
  XLSX <-> JSON
  
  JSON <-> CSV
  
  JSON <-> XLSX
